## What this is

A static portfolio site built without frameworks вЂ” plain HTML, CSS, and vanilla JS. The design direction is "terminal meets editorial": monospace type, a chartreuse accent, and an interactive terminal in the hero where visitors can type commands to explore the content.

The terminal supports: `whoami`, `experience`, `skills`, `projects`, `education`, `contact`, `resume`, `git log`, `man vijay`, `sudo`, and a few easter eggs. Commands auto-scroll to the corresponding section.
